Problems with the site?

Early readers: If you have encountered problems of any kind as you browse the site, please don't hesitate to contact us: andrew.goldstone@rutgers.edu. If the problem appears to be a software or web-browser problem (e.g., the browser doesn't load or behaves strangely), it would be very helpful if you were also able to let us know which browser, which version, and which operating system you are using. But we'd also be grateful to learn of your concerns of any kind relating to the content or the form of this work.

Thank you!
